# ACheckBox
This is a simple CheckBox for Android with cool animation. 


## Import to your project
### Gradle
```
dependencies {
    compile 'com.example.checkbox:checkbox:1.0'
}
```

## Attrs
|attr|format|
|---|:---|:---:|
|stroke_width|dimension|
|color_before|color|
|color_after|color|
|check_text|string|
|check_style|fork/hook|
|show_border|boolean|
